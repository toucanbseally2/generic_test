# generic_test
This is an exercise in creating a generic Functional and Non-functional test suite for cloud applications

Types of Testing up for consideration:

Functional Testing:
  Unit Testing
  Integration Testing
  System Testing
  Sanity Testing
  Smoke Testing
  Interface Testing
  Regression Testing
  Beta/Acceptance Testing

Non-functional:
  Performance Testing
  Load Testing
  Stress Testing
  Volume Testing
  Security Testing
  Compatibility Testing
  Install Testing
  Recovery Testing
  Reliability Testing
  Usability Testing
  Compliance Testing
  Localization Testing
